# My Profile & Links Site

## 📌 概要
自己紹介とリンク集をまとめたシンプルなプロフィールサイトです。  
Webシステムエンジニアを目指す学習の一環として作成しました。

---

## 🌐 公開URL
[（github）](https://main.d14umonfkmzqeq.amplifyapp.com/)

---

## 🧑‍💻 機能
- 自己紹介（名前・アイコン・紹介文）
- 外部リンク（GitHub、X、ブログ）
- レスポンシブ対応（スマホ・PC対応）
- ダークモード切替機能

---

## 🛠 使用技術
- HTML
- CSS
- JavaScript
- Git / GitHub
- AWS Amplify Hosting
